# Документация для сдачи

Эта папка содержит только материалы, которые удобно открыть на защите или показать преподавателю.

- `checklist.md` — полный чеклист по заданиям 2.1-2.6 и дополнительным пунктам.
- `demo.md` — порядок демонстрации приложения и инсталлера.
- `powershell-commands.md` — готовые команды PowerShell для установки, демо, recovery и обхода проблем с правами.
- `answers.md` — короткие ответы на вероятные вопросы преподавателя.

Основной исполняемый файл GUI: `AntivirusWinUi.exe`.

Инсталлер: `AntivirusGuiSetup.exe`.
